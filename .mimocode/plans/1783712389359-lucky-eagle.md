# Plano: Módulo 2 - Exercícios em Vídeo (HTML Trail)

## Contexto

O usuário quer adicionar um novo módulo à trilha HTML da WebStart Academy: "HTML • Módulo 2 - Exercícios em Vídeo". Este módulo contém aulas em formato de vídeo (YouTube embed), diferente das aulas atuais que são baseadas em texto com seções rich (introduction, concept, howItWorks, etc.).

O sistema atual (`Lesson.jsx` + `LessonRenderer.jsx`) é otimizado para aulas de texto com múltiplas seções. As aulas de vídeo exigem um layout diferente: player de vídeo grande no topo, título, descrição, área de aprendizado, materiais, e navegação entre aulas.

## Arquitetura

Criar uma estrutura escalável para aulas de vídeo, sem modificar o sistema existente de aulas de texto:

1. **Dados**: Array de video lessons em um novo arquivo de dados
2. **Página dedicada**: `VideoLesson.jsx` com layout otimizado para vídeo
3. **Rota**: `/video-aula/:lessonId` para distinguir das aulas de texto
4. **Integração**: ModuleDetail.jsx renderiza cards de video lessons corretamente

## Arquivos a Criar/Modificar

### 1. `src/data/video-lessons.js` (CRIAR)
Array de objetos com a estrutura:
```js
{
  id: 'html-video-ex-1',

  moduleId: 'html-video-exercises-module',
  title: 'Exercício 1 - Estrutura Básica em HTML',

  embedUrl: 'https://www.youtube.com/embed/K-MbGXnhGvQ',
  completed: false,
  duration: '',
  order: 1,
  objectives: [...],
  materials: [...]
}
```

### 2. `src/data/lessons/index.js` (MODIFICAR)
Exportar `videoLessons` e funções helper `getVideoLessonById`, `getVideoLessonsByModule`.

### 3. `src/data/modules/html-modules.js` (MODIFICAR)
Adicionar o novo módulo `html-video-exercises-module` no array `htmlModules` com:
- id, courseId ('html'), title, description, order (15)
- lessons: ['html-video-ex-1']
- quiz: null, lab: null, miniProject: null

### 4. `src/data/trails.js` (MODIFICAR)
Adicionar `'html-video-exercises-module'` ao array `modules` da trail `html`.

### 5. `src/pages/VideoLesson.jsx` (CRIAR)
Página dedicada para video lessons com:
- Player de vídeo YouTube responsivo (iframe 16:9)
- Título da aula
- Pequena descrição
- Área "O que você irá aprender" (objectives)
- Área "Materiais da aula" (materials)
- Botão "Marcar como concluída"
- Botões "Aula anterior" e "Próxima aula"
- Barra de progresso da trilha
- Hero section com gradiente (estilo existente)
- SEO component
- Suporte a dark/light mode
- Design system brutalista existente (bordas, sombras, cores)

### 6. `src/App.jsx` (MODIFICAR)
Adicionar rota: `<Route path="video-aula/:lessonId" element={<VideoLesson />} />`

### 7. `src/pages/ModuleDetail.jsx` (MODIFICAR)
Adicionar suporte a video lessons: detectar se a lesson é video ou texto e renderizar o card correto (link para `/video-aula/:id` em vez de `/aula/:id`).

## Detalhes de Implementação

### VideoLesson.jsx - Layout
```
[Voltar para HTML5]
[Hero gradient: "Exercícios em Vídeo" badge + título + descrição]

[Progress bar: X/Y aulas concluídas]

[Card: Player de vídeo YouTube (iframe responsivo 16:9)]

[Título da aula]
[Descrição]

[Card: "O que você irá aprender" - lista de objectives]

[Card: "Materiais da aula" - lista de materials]

[Card: "Concluir Aula" - botão Marcar como concluída]

[Navegação: Aula anterior | Próxima aula]
```

### VideoLesson.jsx - YouTube Embed
```jsx
<div className="relative w-full overflow-hidden rounded-xl border-3 border-strong" style={{ paddingBottom: '56.25%' }}>
  <iframe
    src={lesson.embedUrl}
    title={lesson.title}
    className="absolute inset-0 h-full w-full"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowFullScreen
  />
</div>
```

### Responsividade
- Mobile: player full-width, cards empilhados
- Desktop: layout mais amplo, max-width para legibilidade
- Mesmo padrão do projeto: `p-4 md:p-8`, `max-w-4xl`

### Dados Escaláveis
O array `videoLessons` pode crescer infinitamente. Para adicionar uma nova aula de vídeo, basta adicionar um objeto ao array. O ModuleDetail detecta automaticamente e renderiza o card correto.
